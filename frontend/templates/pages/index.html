CSV Format

Upload your content as a CSV file with the following columns. The system will automatically process and schedule your posts according to the provided information.

Column Description Required
content The text content of your post (max 280 characters) Required
date The date in YYYY-MM-DD format Required
time The time in HH:MM:SS format Required
timezone The timezone (e.g., UTC, America/New_York) Required
thread_id ID to group posts as a thread Optional
thread_position Position of the post in the thread Optional
thread_title Title of the thread Optional
media_urls Comma-separated URLs to media files Optional
