CSV Format

Your CSV file should follow this format:

Column Description
content The text content of your post (max 280 characters)
date The date in YYYY-MM-DD format
time The time in HH:MM:SS format
timezone The timezone (e.g., UTC, America/New_York)
thread_id Optional: ID to group posts as a thread
thread_position Optional: Position of the post in the thread
thread_title Optional: Title of the thread
media_urls Optional: Comma-separated URLs to media files
